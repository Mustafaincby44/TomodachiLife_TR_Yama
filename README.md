# Tomodachi Life: Living the Dream Türkçe Yama (v1.0.3)

Merhaba! 👋

Bu proje, **Tomodachi Life: Living the Dream** oyununun **1.0.3** sürümü için hazırladığım Türkçe yamadır.

Oyunun menülerini, diyaloglarını ve birçok metnini olabildiğince doğal ve akıcı bir Türkçeyle çevirmeye çalıştım. Çeviri boyunca orijinal metne sadık kalmaya özen gösterirken, Türkçede kulağa daha doğal gelen ifadeleri tercih ettim.
